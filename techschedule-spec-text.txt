Coding
Scop: Aplicație web full-stack pentru programarea intervențiilor tehnice (reparații,
instalări, mentenanță), inspirată din fluxul eMag Service. Destinată prezentării ca lucrare
de licență.
1. Viziune Generală
TechSchedule este o platformă care conectează clienții care au nevoie de intervenții tehnice
(reparații electrocasnice, IT, instalații, climatizare etc.) cu tehnicieni specializați. Clientul
programează o intervenție în câțiva pași simpli, tehnicianul primește task-ul în dashboard-ul
lui, iar adminul vede totul dintr-un panou centralizat cu statistici.
Referință UI/UX principală: Pagina de servicii și fluxul de checkout de pe eMag.ro —
header cu search, carduri categorii, stepper de programare cu 4 pași, pagina “Contul meu”
cu tab-uri, status vizual cu linie de progres.
2. Stack Tehnologic
Frontend
React 18 cu Vite ca build tool
Tailwind CSS pentru stilizare
shadcn/ui pentru componente UI (Button, Card, Dialog, Badge, Tabs, Calendar etc.)
React Router v6 pentru navigare
Zustand pentru state management global (user, cart-like appointment flow)
React Query (TanStack Query) pentru fetch și cache date de la API
React Hook Form + Zod pentru validare formulare
date-fns pentru manipulare date calendaristice

from django.db import models

notifications app
class Review(models.Model):
    """
    Review lăsat de client după finalizarea programării.
    Un client poate lăsa un singur review per programare.
    """
    appointment = models.OneToOneField(Appointment, on_delete=models.CASCADE, related_name='review')
    client = models.ForeignKey(User, on_delete=models.CASCADE, related_name='reviews_given')
    technician = models.ForeignKey(Technician, on_delete=models.CASCADE, related_name='reviews_received')
    rating = models.IntegerField()       # 1-5 stele
    comment = models.TextField(blank=True)
    created_at = models.DateTimeField(auto_now_add=True)
    def save(self, *args, **kwargs):
        super().save(*args, **kwargs)
        # Recalculează rating-ul mediu al tehnicianului după fiecare review
        reviews = Review.objects.filter(technician=self.technician)
        avg = reviews.aggregate(models.Avg('rating'))['rating__avg'] or 0
        self.technician.rating = round(avg, 1)
        self.technician.total_reviews = reviews.count()
        self.technician.save(update_fields=['rating', 'total_reviews'])

6. Flux de Programare — Logică Detaliată
Fluxul de programare este feature-ul central al aplicației. Se implementează ca un stepper
cu 4 pași, cu state persistent în bookingStore (Zustand) pe toată durata fluxului.
Pasul 1 — Selectare Serviciu
Se afișează categoriile ca un grid de carduri (icon + nume + descriere scurtă)
La click pe categorie, se expandează sau se navighează la o pagină cu serviciile din acea
categorie
La click pe un serviciu, se salvează selectedService în bookingStore și se trece la
pasul 2
Se afișează un sumar sticky în dreapta (desktop) sau sus (mobile) cu serviciul ales
Pasul 2 — Selectare Locație
Opțiunea A: client selectează una din adresele salvate (liste cu radio button)
Opțiunea B: completează o adresă nouă (formular cu validare) cu opțiunea de a o salva
La submit, se salvează selectedAddress în bookingStore și se trece la pasul 3
Pasul 3 — Selectare Dată și Interval Orar
Se afișează un calendar (componenta shadcn/ui Calendar)
La click pe o dată disponibilă, se face un request la /api/technicians/available-
slots/?date=&service_id=
Backend-ul calculează ce tehnicieni sunt disponibili în ziua respectivă, bazat pe
TechnicianSchedule și programările deja existente

def get_available_slots(date, service_id):
    service = Service.objects.get(id=service_id)
    
    # Găsim tehnicieni cu specialitatea potrivită
    technicians = Technician.objects.filter(
        specialties__services=service,
        is_available=True,
        schedule__day_of_week=date.weekday(),
        schedule__is_working=True
    )
    
    slots = []
    for tech in technicians:
        working_hours = tech.schedule.get(day_of_week=date.weekday())
        booked_slots = Appointment.objects.filter(
            technician=tech,
            scheduled_date=date,
            status__in=['pending', 'confirmed', 'in_progress']
        ).values_list('time_slot_start', 'time_slot_end')
        
        # Generează sloturi de 2h din intervalul de lucru, excluzând cele ocupate
        available = generate_free_slots(
            working_hours.start_time,
            working_hours.end_time,
            service.duration_min,
            booked_slots
        )
        
        for slot in available:
            slots.append({
                'technician_id': tech.id,
                'technician_name': tech.user.get_full_name(),
                'start': slot['start'],
                'end': slot['end']
            })

        set({ user, accessToken, refreshToken }),
      logout: () => set({ user: null, accessToken: null, refreshToken: null }),
      refreshToken: async () => {
        const refresh = get().refreshToken;
        if (!refresh) return false;
        try {
          const res = await fetch('/api/auth/token/refresh/', {
            method: 'POST',
            headers: { 'Content-Type': 'application/json' },
            body: JSON.stringify({ refresh }),
          });
          if (res.ok) {
            const data = await res.json();
            set({ accessToken: data.access });
            return true;
          }

        return false;

  { key: 'confirmed', label: 'Confirmat', icon: CheckCircleIcon },
  { key: 'in_progress', label: 'În lucru', icon: WrenchIcon },

];
export function AppointmentTimeline({ status }) {
  const currentIndex = STEPS.findIndex(s => s.key === status);
  return (
    <div className="flex items-center justify-between w-full">
      {STEPS.map((step, index) => {
        const isDone = index <= currentIndex;
        const Icon = step.icon;
        return (
          <div key={step.key} className="flex flex-col items-center flex-1">
            <div className={`rounded-full p-2 ${isDone ? 'bg-blue-600 text-white' : 'bg-gray-200 text-gray-400'}`}>
              <Icon className="w-5 h-5" />
            </div>
            <span className={`text-xs mt-1 ${isDone ? 'text-blue-600 font-medium' : 'text-gray-400'}`}>
              {step.label}
            </span>
            {/* Linie de conectare între pași */}
            {index < STEPS.length - 1 && (
              <div className={`absolute h-0.5 w-full ${isDone ? 'bg-blue-600' : 'bg-gray-200'}`} />
            )}
          </div>
        );
      })}
    </div>
  );
}

export function ProtectedRoute({ children, requiredRole }) {
  const { user } = useAuthStore();
  const location = useLocation();
  if (!user) {
    return <Navigate to="/login" state={{ from: location }} replace />;
  }
  if (requiredRole && user.role !== requiredRole) {
    // Redirect la dashboard-ul corespunzător rolului real
    const dashboardMap = {
      client: '/client/dashboard',
      technician: '/technician/dashboard',
      admin: '/admin/dashboard',
    };
    return <Navigate to={dashboardMap[user.role]} replace />;
  }
  return children;
}
